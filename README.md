## ğŸš€ Getting Started

### View Locally

1. Clone the repository:
```bash
git clone https://github.com/wclaytor/wclaytor.github.io.git
cd wclaytor.github.io
```

2. Open `index.html` in your browser or use a local server:
```bash
# Python 3
python -m http.server 8000

# Node.js (with http-server)
npx http-server
```

3. Visit `http://localhost:8000`

### Development

The site uses vanilla HTML, CSS, and JavaScript with Alpine.js loaded from CDN. No build process required for basic development.

Automatically deployed to GitHub Pages when pushing to the `main` branch.
